# HAMAS - A Distributed Agent-based Energy Management System
## Installation
### System Installation
In this case you need root or admin permissions.

```bash
# cd /path/to/project/root
# pip install -e .
```
### User Installation
In case you don't have root or admin permissions.

```bash
$ cd /path/to/project/root
$ pip install --user -e .
```
